# СКАН
<br>
Сервис по поиску публикаций в СМИ о компании ИНН.<br>
<br>
Training project.<br>
<br>
Technologies and libraries:<br>
- HTML<br>
- CSS Modules<br>
- <b>React</b><br>
- <b>Redux</b><br>
- <b>React Router</b><br>
- Redux Persist<br>
- React Hook forms<br>
- React Slick Slider (all sliders)<br>
- MUI (Datepicker)<br>
- Axios<br>
<br>
etc.<br>
